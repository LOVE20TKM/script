钱包地址时间线看板

输入任意钱包地址后，页面会从专用看板 SQLite 里按区块和交易顺序拉出该地址的时间线，并把 `ClaimReward`、`groupJoin`、`Approval`、转账、加池 / LP 铸造这类行为归类成接近人工整理的表格。

时间线主真相源是 events、transactions、blocks。页面显示的是基于这些记录归纳后的交易级摘要，不直接查询 6G 源库，因此需要先运行当前目录下的 refresh.sh 生成专用看板 DB。
